# Dayu 功能对照表

状态：规划基线

## 使用方式

Dayu 是公开参考产品，用于确认龙鳞装创作工具的功能链路和用户预期。本项目只对照功能，不复制 Dayu 的代码、视觉、品牌、密钥、付费流程或工程文件格式。

优先级说明：

- P0：MVP 必须完成。
- P1：本地编辑和导出稳定后完成。
- P2：展示、录制、模板等增强能力。
- P3：商业化和云端能力，单独审批。
- 不做：当前路线明确排除。

## 当前 UI 重构验收

第一版中文界面必须体现 Dayu 主流程，但不需要完成所有功能：

- 顶部有项目、导入、保存、导出、预览、帮助入口。
- 左侧是素材栏：正面长卷、背面长卷、内页叶片。
- 中间是二维装帧画布。
- 右侧是装帧参数、推导尺寸和检查结果。
- 所有用户可见文案使用中文。
- 未实现功能使用占位或禁用态，不能伪装成已完成。
